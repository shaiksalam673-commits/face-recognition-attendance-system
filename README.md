# Face Recognition Attendance System

This project is a real-time face recognition-based attendance system built using OpenCV and Python.

## Features
- Face detection using Haar Cascade
- Face recognition using LBPH algorithm
- Automatic attendance marking with timestamp
- CSV-based storage system

## Technologies Used
- Python
- OpenCV
- NumPy

## How to Run
1. Run capture.py to collect images
2. Run train.py to train the model
3. Run recognize.py to mark attendance
